Loan Calculator Suite — 7 языков, полный расчёт кредитов
Loan Calculator Suite — коллекция из семи независимых реализаций кредитного калькулятора, каждая из которых работает на своём языке программирования. Проект создан для демонстрации кроссплатформенного подхода и предоставления удобного инструмента для расчёта параметров кредита с расширенными возможностями.

✨ Уникальные возможности (общие для всех версий)
💰 Расчёт аннуитетных и дифференцированных платежей.

📅 Детальный график платежей с разбивкой по месяцам (номер, остаток долга, основной долг, проценты, общий платёж).

📊 Итоговые показатели: общая сумма выплат, переплата, эффективная процентная ставка (приближённо).

⚙️ Гибкая настройка: сумма кредита, годовая ставка, срок (в месяцах), тип платежа.

🧾 Экспорт графика в CSV и TXT.

🖥️ Интерфейсы:

Командная строка (CLI) с параметрами и интерактивным режимом.

Графический интерфейс (GUI) в Python, Java, C#.

Веб-интерфейс в PHP (и опционально в JavaScript).

📈 Визуализация (в некоторых версиях) — цветное выделение в консоли.

Использование (общий синтаксис)
CLI (все версии):

bash
loan_calc --amount 1000000 --rate 12 --term 60 --type annuity --schedule
Пример:

bash
python loan_calc.py --amount 1000000 --rate 15 --term 24 --type diff --output schedule.csv
Интерактивный режим (CLI):
Запустите без аргументов – откроется меню.
